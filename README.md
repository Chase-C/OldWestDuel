Gun Factor (Old West Duel)
===============

A two player game created for CodeDay SV 2014.  
The game can be played online [right here](http://chase-c.github.io/OldWestDuel/)!

Controls
========

Player 1
--------
w - Jump  
s - duck  
f - shoot  

Player 2
--------
up - jump  
down - duck  
'.' - shoot
